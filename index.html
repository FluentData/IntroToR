Getting Started with R

This course is designed to help air quality analysts get started using R. No previous experience with R or any other programming language is required. In this course you will learn what R is, how to install it, and how to use the command line. You will also learn how to install and use RStudio. The course will cover the basics about the R language, such as data types and functions. You will learn how to import data and do basic data manipulation, like subsetting and sorting. The training also shows you how to do basic statistical analysis on air quality data and how to visualize your data in plots.

  • Subsetting, Sorting, and Dates
  • Writing Functions, Conditionals, and Loops
